To run the C version of the shop navigate to ".\C\" using cygwin.
With cygwin run "gcc shop.c" to compile the code.
Lastly running ".\a.exe" to run the executable.

To run the python version of the above, navigate to ".\Python Procedural\".
To start the program execute "python shop.py".

Lastly to run the object-oriented version of the shop, navigate to ".\Python OOP\".
To start the program execute "python shop_oop.py".

Customer csv data files can be found in ".\Customer Orders\".
Shops csv data file can be found in ".\Shop Stock\".
